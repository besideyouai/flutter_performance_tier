# 文档导航

`docs/` 只保留少量长期维护的文档。

当前状态、下一步和收口事项统一维护在 `plan/development_plan.md`，不再单独拆分日期型的状态页或审查页。

## 优先阅读

- `plan/development_plan.md`：项目目标、当前阶段、下一步与验收口径。
- `plan/real_device_acceptance_checklist.md`：真机验收 checklist 与记录模板。
- `progress/runtime_dynamic_tiering.md`：运行期动态降级规则、联调模板与测试说明。

## 补充参考

- `progress/initialization_baseline.md`：初始化耗时基线与测量口径。
